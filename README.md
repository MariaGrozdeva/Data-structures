## Реализация на *абстрактни* структури от данни:
**1.Стек**- Добавяне и премахване на елемент от една и съща страна.

|  pushBack |   |  popBack |  
----------------------------
| Вектор  |  O(1) | O(1)  |  
|  Едносвързан списък | O(1)  | O(n)  |  
| Двусвързан списък  | O(1)  |  O(1) |
